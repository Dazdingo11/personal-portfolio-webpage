Projects

IwiseAI

An AI automation platform built to replace office roles safely and reliably. Core loop mirrors real knowledge work: Input → Identify → Plan → Act → Record.

  • Agents: Safety, Classifier, Policy, Planner, Critic, Executor, Watchdog, Memory, Evaluator, Supervisor.
  • Compliance: GDPR, PII redaction, policy packs, tenant scoping & audit logs.
  • HRM brain: knowledge + permissions + tools; plans tasks and executes within allowed scopes.
  • Modes: Safe (analysis-only), Learn (human-in-loop), Production (autonomous).

RankedWithAI

An IMDB-style ranking platform for AI tools built with Next.js. Browse fast carousels and card grids, discover by category or tag, and surface community signals.

  • Responsive cards & carousels for quick skimming.
  • Categories, tags, search; sort by rating/new/trending.
  • Static generation with revalidation for speed.

SmartPriceTracker

A lightweight price-tracking toolkit in Python that exports clean CSV/XLSX. Designed to run locally with fallbacks so demos work even when sites add anti-bot barriers.

  • Modes: Demo (local HTML), HTTP (Requests + BeautifulSoup), Selenium (dynamic sites).
  • Reliability: timeouts, retries, fallbacks to local parsing; VPN support to improve stability.
  • Normalized exports: name, price, source, url.

King’s Barbers

A minimal, responsive landing page for a local barbershop. HTML/CSS with small JS for UI. Prototype Python FastAPI backend (auth + bookings); GitHub Pages demo uses a static mock.

  • Sections: hero, services, gallery, pricing, contact.
  • Deployment: GitHub Pages (/docs on main).
